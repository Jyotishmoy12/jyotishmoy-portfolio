# An Algorithm for the Longest Common Subsequence and Substring Problem

**Journal:** Journal of Mathematics and Informatics, Vol. 25, 2023, 77-81  
**Authors:** Rao Li, Jyotishmoy Deka, and Kaushik Deka

---

## Abstract
The Longest Common Subsequence (LCS) and Longest Common Substring (LCStr) problems are fundamental in string processing. This 2023 paper introduces a novel algorithm to address these problems efficiently, building the foundation for the constrained version published in 2024.
